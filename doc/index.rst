=====================
 Etude C++ Libraries
=====================

Etude C++ Libraries は， C++0x の汎用小物ライブラリです．

主に，標準ライブラリや `Boost C++ Libraries <http://boost.org/>`_ を使用する際に必要となる ちょっとした部品を提供することで，ただでさえ便利な C++0x を，更に便利にすることを目的としています．

具体的なライブラリ内容は， :doc:`/libs` を参照してください．
